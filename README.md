#☕ Java Café Billing System ☕

A simple Java console application that allows users to order items from a café menu, calculates the total bill, and converts prices from USD to INR.

## Features
- Menu with Coffee, Tea, Sandwich, and Cake
- Price conversion from USD to INR (₹83 rate)
- Quantity-based billing
- Continues ordering until the user exits
- Displays final total bill

## How to Run
1. Compile the program:
   javac shop.java

2. Run the program:
   java shop

## Technologies Used
- Java
- Scanner Input
- Arrays
